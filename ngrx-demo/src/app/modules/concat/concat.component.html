Concat operator Example

Whwn we have to collect data from diffrent stream and show it in single output field then we use concat operator to handle such sitation. It generally used when we are strict toward the sequence of data and this sequnce matters a lot . it simply subscribe data of one observation and once it finish only then it start subscribing other observation data.

Ex -01

Animals ==> ['tiger','Bear','Lion','deer','zebra','fox']

Ex -02

Bird ==> ['pigeon','dove','cukoo','eagle','vulture','penguin']

Ex -03

Food ==> ['pizza','samosa','burger','idli','dosai','litti']

Here we can see the data coming from diffrent Stream are in serial manner and when one stream data completed then only another stream data is subscribed.
